Що таке CMS

CMS - "система керування контентом" (движок) - написана PHP-програмістами основа для сайту, за допомогою якої ви зможете керувати сайтом (додавати контент, змінювати пункти меню тощо) не знаючи HTML та CSS.

Однак, для того, щоб зробити сайт за допомогою CMS будуть потрібні послуги і програміста, і дизайнера, і верстальника. І капіталовкладення.

Які бувають cms

Бувають різні системи управління контентом: для інтернет-магазинів, блогів, форумів і т.д.

Приклади cms

Приклади найпопулярніших CMS: Joomla, WordPress (для блогів), PhpBB (для форумів).

CMS-ки бувають платні та безкоштовні.

PHP и JavaScript

Мови програмування PHP і JavaScript дозволяють зробити сайт динамічним, тобто таким, що реагує на дії користувача. Наприклад, можна зробити красиву меню, що випадає, або слайдер.

Види скриптів

Для цього пишуться скрипти (англ. script – "сценарій") – програми, що дозволяють реагувати на дії користувача. Скрипти бувають двох видів:

  • ті, які виконуються на сервері, а результат їхнього виконання приходить у браузер до користувача вже в готовому вигляді. Це скрипти, написані мовою PHP. На ньому пишуться CMS-ки - системи управління контентом.
  • ті, які виконуються прямо у браузері користувача. Це скрипти, написані мовою JavaScript. Вони найчастіше використовуються для того, щоб зробити сторінку більш зручною та красивою.
